The kickstart file is a file that answers the questions asked during a normal installation of ubuntu. One can use such a file to preform an automated installation. To use the file put it on a (local) web or ftp server and remember the ip address. Then start the machine that you want to preform the installation on with the installation media already connected.
When you see a menu like the one below press the ⇥ Tab key.

In the menu entry append ks=http://example.com/ks.cfg with example.com replaced by your ip address. Then press ⏎ Enter and wait until it finishes, asks a question or fails. If it fails your file is likely incomplete in a way it cannot ask you what it is missing.

For more information see: https://help.ubuntu.com/community/KickstartCompatibility and https://help.ubuntu.com/lts/installation-guide/i386/ch04s06.html
